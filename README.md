# Web3 Liquidity Pool Monitor

A professional Web3 reference project that demonstrates how decentralized exchange liquidity pools can be modeled and analyzed using clean and modular Web3 architecture.

This repository focuses on simulating pool reserves, tracking swap activity, monitoring pool health metrics, and generating analytics reports in a beginner-friendly and production-style format.

---

## Supported Networks

- Ethereum  
- Base  
- Polygon  
- Arbitrum  

---

## Project Goals

- Monitor liquidity pool reserves  
- Simulate swap transactions  
- Track pool health metrics  
- Analyze volume and activity  
- Teach DEX monitoring architecture  

---

## Architecture Overview

This project uses a root-only structure where each file handles a single responsibility such as network access, pool registry, swap simulation, health analysis, and analytics reporting.

---

## Features

- Multi-chain RPC support  
- Pool reserve simulation  
- Swap activity tracking  
- Pool health analytics  
- Clean console-based reporting
